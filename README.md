 Sales & Cost Analysis Dashboard (Power BI)
This project presents an interactive Sales & Cost Analysis Dashboard built using Power BI, designed to provide actionable insights into business performance across multiple dimensions.
 Overview
The dashboard enables users to explore and analyze key metrics such as cost, revenue (EUR), sales performance, and regional trends. It is designed to support data-driven decision-making through intuitive and interactive visualizations.
 Key Features

Order-Level Cost Analysis
Identify high-cost transactions and understand cost distribution across orders.


Sales Representative Performance
Compare contributions of different sales reps across multiple orders.


Category-Wise Insights
Analyze trends between cost and revenue (€) across product categories.

Geographical Analysis
Explore performance across countries with dynamic filtering capabilities.

Device-Type Breakdown
Understand customer behavior based on device usage (Mobile, Tablet, PC).

Sales Manager Contribution
Visualize revenue share across sales managers for performance evaluation.

Interactive Filtering
Slice data by country and other dimensions for deeper insights.
